CSS Border Types Demonstration

Solid Border: This box uses a solid black border which is the default style for borders in CSS.
Dotted Border: This box uses a blue dotted border. Each side of the border is made of dots.
Dashed Border: This box has a red dashed border, which appears as broken lines.
Double Border: This green border is double-lined, giving a bold look.
Groove Border: This purple groove border gives a 3D effect as if the content is carved in.
Ridge Border: This orange ridge border gives a raised 3D effect to the box.
